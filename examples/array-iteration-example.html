Array Iteration Examples

Basic Array with Comma

Contact us: [[@array(contacts),]]

Custom Separators

Contact us: [[@array(contacts) | ]]

Contact us: [[@array(contacts) • ]]

Contact us: [[@array(contacts) - ]]

No Separator (Space Default)

Contact us: [[@array(contacts)]]

Nested Array

Organizers: [[@array(event-data.organizers),]]

Sponsors: [[@array(event-data.sponsors) | ]]

Complex Object Array

Speakers: [[@array(speakers),]]

Speaker Names: [[@array(speakers.name),]]

Speaker Titles: [[@array(speakers.title) | ]]

Nested Arrays with Objects

Organizers: [[@array(event-data.organizers),]]

Sponsors: [[@array(event-data.sponsors) | ]]

Speaker Names from Event Data: [[@array(event-data.speakers.name),]]

Empty Array

No contacts: [[@array(empty-contacts),]]

Internationalization Examples

English: [[@array(contacts),]]

Japanese: [[@array(contacts),]]

Chinese: [[@array(contacts),]]

Japanese with proper separator: [[@array(contacts) 、]]

Chinese with proper separator: [[@array(contacts) ，]]
